# COISA-drivetrain-calculator
C.O.I.S.A — Code fOr calculatIon of uSing a drivetrAin (yes the order of the words in the phrase dont make sense but its funny in my native lang.). A gear ratio analyzer for brushed DC line follower robots. Calculates speed, torque, slip and current draw across PWM points, ranks gear combinations by score, and exports results to Excel.
